# 画布编辑器功能清单

## ✅ 已实现功能

### 1. 数据解析
- [x] JSON自动解析为树形结构
- [x] 显示字段类型（Object、Array、String、Number等）
- [x] 支持嵌套结构解析
- [x] JsonPath路径生成（源数据使用$前缀）

### 2. 拖拽功能
- [x] HTML5拖拽API实现
- [x] 从源数据树拖拽到画布
- [x] 从目标数据树拖拽到画布
- [x] 节点在画布上可自由移动

### 3. 画布功能
- [x] AntV X6图编辑引擎集成
- [x] 节点创建和显示
- [x] 连线创建（源节点到目标节点）
- [x] 画布缩放（鼠标滚轮+Ctrl）
- [x] 画布平移（拖拽背景）
- [x] 节点删除（Delete键）
- [x] 连线删除（Delete键）
- [x] 自动删除相关连线（节点删除时）

### 4. 连线配置
- [x] 点击连线打开配置对话框
- [x] 映射类型选择（1对1、1对多、多对1）
- [x] 转换类型选择（直接、函数、Groovy、字典、固定值）
- [x] 函数映射配置（选择预置函数）
- [x] Groovy脚本配置（代码编辑器）
- [x] 字典映射配置（键值对配置）
- [x] 固定值配置
- [x] 配置保存和更新

### 5. 数据导出
- [x] 导出MappingConfig格式
- [x] 遍历所有连线和节点
- [x] 提取源路径和目标路径
- [x] 提取转换配置
- [x] 复制到剪贴板

### 6. 测试转换
- [x] 调用后端API
- [x] 传递源数据和配置
- [x] 显示转换结果
- [x] 自动更新目标数据区域

### 7. UI/UX
- [x] 三栏布局（源数据、画布、目标数据）
- [x] 源节点样式（蓝色系）
- [x] 目标节点样式（紫色系）
- [x] 连线标签显示转换类型
- [x] 画布提示文字
- [x] 响应式设计
